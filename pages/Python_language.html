Python !!

Python à été créer en 1991, il est utilisé partout et pour tous. En développement web, application ,script même chez Netflix.

Utilisation

Il permet la creation d'application web pour les serveurs, contrôler les fulx de données (workflows). Peut se connecter à une base données, lire et modifier les fichiers. Utilisé en Big Data, il peut accomplir des calculs extrêmement complex.

Syntaxe simple et concise. Prototypage ultra rapide.

Coder en Python

Au contraire des autres langages pour coder en Python, vous devrez l'installer en premier. Et ensuite vous pourrez choisir un éditeur qui répond à vos besoins. Une fois votre code écrit vous devrez le compiler, pour ensuite pouvoir le tester. Certain editeur et site web rassemble tous les outils au même endroit.
