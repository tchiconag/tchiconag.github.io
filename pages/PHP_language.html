Le PHP

Créé en 1994, le PHP est un langage de script pour serveur et un outil puissant pour créer des pages web dynamiques et interactives. Il est largement utilisé sur le marché, le PHP7 est la derniere version stable du langage.

Utilisation

Le PHP vous permettra de créer des pages qui seront générées dynamiquement. Donc, grâce au PHP, vous porrez afficher des contenus différents sur une même page en fonction de certaines variables : La datte, l'heure ,utilisateur connu ou non, connecté ou pas, etc.

Les éditeurs de text (IDE)

Plusieurs éditeurs existe aussi pour le PHP, les IDE sont des tous en un qui permettent d'écrire, compiler et tester son code . Tandis les éditeurs de code sont des éditeurs de text avec des fonctions qui facilite l'écriture.
